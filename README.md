Описание
========================
В данном приложении реализованы поиск треков, создание плейлистов и добавление в них песен. Есть избранное, базовые настройки, реализованы стандартные действия над плейлистами, такие как поделиться, редактировать и удалить. Приложение создано в рамках прохождения курса Android-разработчик от Яндекс Практикума.

Технологии
========================
* Kotlin
* Retrofit
* Glide
* CLEAN
* DI
* ViewModel
* LiveData
* SharedPreferences
* ViewBinding
* Koin
* Fragments
* Jetpack navigation component
* Bottom navigation
* Сoroutines
* SQLite
* ROOM
